## 🚀 Deploy to GitHub Pages

1. **Create a GitHub repository** (name it `piperine` or your preferred name)

2. **Upload all files** maintaining this folder structure:
   ```
   git init
   git add .
   git commit -m "Initial deploy — Piperine nursery website"
   git remote add origin https://github.com/YOUR_USERNAME/piperine.git
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to your repo → Settings → Pages
   - Source: `Deploy from a branch`
   - Branch: `main` / `(root)`
   - Click Save

4. **Your site will be live at:**
   `https://YOUR_USERNAME.github.io/piperine/`
